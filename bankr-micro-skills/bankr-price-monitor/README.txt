BANKR PRICE MONITOR SKILL
Autonomous Token Price Tracking & Conditional Trading

This skill enables autonomous AI agents to:
- Monitor token prices across Base, Ethereum, Polygon, Solana
- Set price-based trading triggers (buy if <$X, sell if >$Y)
- Execute swaps automatically via Bankr
- Run multiple strategies simultaneously
