# Polymarket Agent Dashboard

> Real-time dashboard to track autonomous Polymarket trading agent performance

![Dashboard](https://img.shields.io/badge/Status-Active-brightgreen)
![License](https://img.shields.io/badge/License-MIT-blue)

## 🚀 Features

- **Live Equity Tracking** - Watch your portfolio grow in real-time
- **Performance Analytics** - Win rate, ROI, P&L metrics
- **Trade History** - Complete log of all executed trades
- **Charts & Visualization** - Equity curves and win/loss distribution
- **Zero Security Risk** - No API keys in the frontend
- **Easy Deployment** - One-click deploy to Vercel

## 📊 Dashboard Preview

The dashboard displays:
- Current equity and bankroll
- Total return percentage
- Win rate across all trades
- Equity curve chart
- Win/loss distribution pie chart
- Detailed trade history table

## 🔒 Security

**Your Bankr API key is NEVER exposed:**

```
Your VPS (Private)
    ↓
    Agent executes trades (uses API key)
    ↓
    Writes performance log (agent-data.json)
    ↓
Vercel (Public)
    ↓
    Fetches the public log
    ↓
    Renders dashboard (no credentials needed)
```

## 🛠️ Setup

### Prerequisites

- Node.js 18+
- Vercel account (free)
- GitHub account (optional, for CI/CD)

### 1. Clone & Install

```bash
git clone https://github.com/yourusername/polymarket-dashboard.git
cd polymarket-dashboard
npm install
```

### 2. Local Development

```bash
npm run dev
```

Visit `http://localhost:3000`

### 3. Deploy to Vercel

```bash
npm i -g vercel
vercel --prod
```

**Or connect to GitHub:**
1. Push to GitHub
2. Go to https://vercel.com
3. Import your repo
4. Deploy (auto-deploys on push)

### 4. Configure Agent Data Source

The dashboard looks for agent data at `public/agent-data.json`.

**On your VPS**, after agent runs, ensure this file exists:
```bash
/home/openclaw/.openclaw/workspace/polymarket-agent/agent-data.json
```

Example format:
```json
{
  "lastUpdate": "2026-02-24T10:30:00Z",
  "bankroll": 12.50,
  "totalWagered": 25.00,
  "totalWon": 5.00,
  "totalLost": 1.25,
  "trades": [
    {
      "id": "trade_1",
      "timestamp": "2026-02-24T10:00:00Z",
      "market": "Will BTC reach $100k?",
      "outcome": "Yes",
      "amount": 2.50,
      "status": "closed",
      "entryPrice": 0.375,
      "exitPrice": 0.45,
      "pnl": 0.1875,
      "pnlPct": 0.25
    }
  ],
  "equity": [10, 10.25, 12.50]
}
```

## 🔄 How It Works

1. **Agent runs on your VPS** (every 2 hours via cron)
2. **Agent logs trades** to `agent-data.json`
3. **Dashboard fetches** the JSON file
4. **Charts update** with latest data
5. **Anyone can view** (password-protected coming soon)
